## ✨ 使用方式：

1. 安装浏览器扩展 [Stylus](https://add0n.com/stylus.html)  
2. 点击主题文件（.user.css）→ 点击「Raw」 → 复制内容  
3. 粘贴到 Stylus 新建样式中，设置为匹配 ChatGPT 网页域名：

```txt
https://chat.openai.com/*
https://chatgpt.com/*
```
